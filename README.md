# AlphaVest — AI Trading Signals (Powered by Groq AI — Free & Fast)

## 🆓 Get FREE Groq API Key (No Credit Card)

1. Go to **https://console.groq.com**
2. Sign up with Google or email
3. Click **API Keys** → **Create API Key**
4. Copy the key (starts with `gsk_...`)
5. Done — completely free, very fast

## 🚀 Deploy to Vercel

1. Push to GitHub
2. Import in Vercel
3. Add Environment Variable:
   - **Name:** `GROQ_API_KEY`
   - **Value:** `gsk_...`
4. Deploy ✓
